# LHB HR Management System

HR Management System for Lavish Home Build 丰盛建筑

## Features
- Staff Management
- Daily Attendance (QR Code Scan)
- Staff Leave Management  
- OT Management
- Payroll Calculation
- Pay Slip Generation

## Setup
1. Open `index.html` in browser
2. Configure Google Sheet URL in Settings
3. Login with admin credentials

## Access
- **PC**: Open index.html directly
- **Mobile**: Visit GitHub Pages URL

---
Built with Google Apps Script + Vanilla JS
